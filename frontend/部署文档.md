### 3.3 应用服务器部署

#### 3.3.1 部署描述

部署前端Vue程序，执行以下步骤：

 - 安装Node.js和npm。
 - 克隆代码库。
 - 安装依赖项：运行 `npm install`。
 - 配置环境变量。
 - 项目打包：运行`npm run build`
 - 将项目中的`dist`文件夹提交到服务器上。

部署后端Spring boot项目，执行以下步骤：

- 克隆代码库
- 配置依赖项
- 项目打包：运行`mvn clean package -DskipTests`
- 将生成的`jar`包提交到服务器上
- 执行命令`nohup java -jar XXX.jar &`以持续运行。
